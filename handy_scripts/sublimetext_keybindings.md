# Keybindings

Hereâ€™s a details of the keybindings:

1. **Multiple Cursors**:
   - **Windows**: `Command + Click` to place multiple cursors.
   - **Mac**: `Command + Click` to achieve the same effect.

2. **Modifying Selections**:
   - **Windows**: `Ctrl + D` to select the next occurrence of a word.
   - **Mac**: `Command + D` for the same function.

3. **Multi-Line Cursors**:
   - Similar to multiple cursors, use `Ctrl + Click` (Windows) or `Command + Click` (Mac).

4. **Line Manipulation**:
   - **Combine Lines**: `Ctrl + J` (Windows) / `Command + Shift + J` (Mac) to join lines.
   - **Select Entire Line**: `Ctrl + L` (Windows) / `Command + L` (Mac).
   - **Delete or Cut a Line**: `Ctrl + X` (Windows) / `Command + X` (Mac).
   - **Move Line Up/Down**: `Ctrl + Shift + Up/Down Arrow` (Windows) / `Command + Control + Up/Down Arrow` (Mac).
   - **Duplicate a Line**: `Ctrl + Shift + D` (Windows) / `Command + Shift + D` (Mac).
   - **Go to Specific Line**: `Ctrl + G` (Windows) / `Ctrl + G` (Mac).

5. **Indentation**:
   - **Indent**: `Ctrl + Right Bracket` (Windows) / `Command + Right Bracket` (Mac).
   - **De-indent**: `Ctrl + Left Bracket` (Windows) / `Command + Left Bracket` (Mac).

6. **Selecting Contents Within Parentheses**:
   - **Selection**: `Ctrl + Shift + M` (Windows) / `Ctrl + Shift + M` (Mac) to select contents within brackets.

7. **File Navigation and Search**:
   - **Open File**: `Ctrl + P` (Windows) / `Command + P` (Mac) to quickly open files.
   - **Search in Files**: `Ctrl + Shift + F` (Windows) / `Command + Shift + F` (Mac) to search through all files in the workspace.

8. **Column Layout**:
   - To create multiple columns, use `Alt + Shift + Number` (Windows) / `Option + Shift + Number` (Mac).

Ref: [Sublime Text Keyboard Shortcuts](https://docs.sublimetext.io/reference/keyboard_shortcuts_osx.html)
